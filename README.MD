### how to using it in your project

- install vm tools
```shell
sealvm install 
```

- init vms 
```shell
sealvm run --nodes=1 
```

- init vms add dev 
```shell
 sealvm run --nodes=1 --dev --dev-mounts /Users/cuisongliu/Workspaces/go/src/github.com/labring/sealos:/root/go/src/github.com/labring/sealos 
```

- reset vms
```shell
sealvm reset
```
